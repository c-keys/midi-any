# MIDI Any v1

![](https://ckeys.org/images/midi-any-mockup-banner.jpg)

## Slides

Slides can be found at: https://ckeys.org/slides/midi-any/

## Firmware

If you want to flash new firmware, use a `.bin` file for the Proton C. Flash with the [QMK Toolbox](https://docs.qmk.fm/#/newbs_flashing) or via the command line.

If you would like to modify the firmware, the MIDI Any keymaps will soon be available in the main [QMK github repo](https://github.com/qmk/qmk_firmware). However, the MIDI Any firmware is actively being improved and so temporarily you can access the keymap files on [a temporary fork of the QMK project](https://github.com/brandenbyers/qmk_firmware/tree/midi-any).

If you have any questions about the firmware or how to modify it, please [contact us](https://ckeys.org/contact) and we will be happy to help.
